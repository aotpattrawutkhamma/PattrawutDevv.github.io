About My Work Application

งานของผมจะพัฒนาในส่วนของ Mobile Application โดยใช้ภาษา Flutter

ผมได้ทำงานในตำแหน่ง Programer โดยจะทำงานเกี่ยวกับการพัฒนา Mobile Application เป็นหลัก โดยโปรแกรมที่ผมพัฒนาจะเป็น Application ที่ใช้ภายในองค์กร เท่านั้น เช่น AMS,CountStock,Inventory โดยในระบบ จะทำงานได้ทั้ง Offline และ Online โดยใช้ Flutter ในการพัฒนา และใน Sqlite,Drift ในการจัดการข้อมูล และสามารถ Exportเป็น Excel ได้ ในบางระบบ จำเป็นต้องเชื่อมต่อกับตัวเครื่อง RFID ซึ่งผมได้ทำการเชื่อมต่อกับ RFID ได้ โดยผ่าน SDK ของ RFID ยี่ห้อนั้นๆ

**เนื่องจากเป็น Application ภายในองค์กรที่มีการขายและใช้งานอยู่ จึงไม่สามารถเปิดเผยรายละเอียดได้ ทั้งหมดใน Git จะเป็นเพียงส่วนหนึ่งของโค้ดที่สามารถเปิดเผยได้ ที่ผู้จัดทำเห็นว่าไม่มีปัญหาในการเปิดเผย และเป็นตัว Demo

About My Work Website And API

งานของผมจะพัฒนาในส่วนของ Website และ API((Swagger)) โดยใช้ภาษา .net ในการพัฒนา

โดยส่วนใหญ่งานของ .NET จะเป็นการพัฒนา API ที่ใช้ในการเชื่อมต่อกับ Mobile และการ support issue ที่เกิดขึ้นในระบบ จะใช้ entity framework ในการเชื่อมต่อกับฐานข้อมูล และการเขียน View และ store procedure ในการดึงข้อมูล

**เนื่องจากเป็น Application ภายในองค์กรที่มีการขายและใช้งานอยู่ จึงไม่สามารถเปิดเผยรายละเอียดได้ ทั้งหมดใน Git จะเป็นเพียงส่วนหนึ่งของโค้ดที่สามารถเปิดเผยได้ ที่ผู้จัดทำเห็นว่าไม่มีปัญหาในการเปิดเผย และเป็นตัว Demo
